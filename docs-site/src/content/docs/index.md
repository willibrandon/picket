Picket is a MIT-licensed secrets scanner designed around Gitleaks compatibility, Native AOT distribution, and byte-oriented matching through Scout libraries.

Use this site for generated project documentation, command reference, GitHub Action reference, and public API reference as the implementation matures.

## Documentation Model

- Durable project documentation lives in `docs/`.
- Generated site pages are produced by `tools/Picket.Docs`.
- The published site is static and deploys to GitHub Pages.
- No custom server, custom domain, live scanner, or hosted validator is required.
